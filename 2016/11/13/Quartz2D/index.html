Quartz2D绘图

基本介绍

核心绘图! CoreGraphics -> graphics 图形,图像
是CoreGraphics 框架下的内容 C语言框架!
函数大都以CG开头!

注意: C语言创建的结构体 不需要加 *;
核心绘图,在iOS和Mac OS 两个系统系统里面都可以用

目的:在应用中,可以根据数据实时的更新图像!

基本绘图步骤

  • 1.重写视图的drawRect方法
  • 2.获取图形上下文 -> 绘图的环境
  • 3.创建想要绘制的图形的路径 -> 几何图形!
  • 4.将路径添加到环境上!
  • 5.渲染 -> 将图形生成图像显示到屏幕上!

填充规则介绍

非零绕数填充规则

kCGPathFill
视图中的一个点,
被顺时针覆盖一次, 标记为 1
被逆时针覆盖一次, 标记为 -1

最终结果只要是 0 ,就不填充! 否则,填充!

奇偶填充规则

kCGPathEOFill
视图中的一个点
如果被覆盖奇数次,在渲染的时候填充!
如果被覆盖偶数次,在渲染的时候不填充!

drawRect方法介绍

什么时候调用?

第一次显示视图的时候,会调用一次!

参数rect代表的含义?

  • 默认就是视图自己bounds

为什么要重写这个方法?

  • 因为苹果要求在此方法中写绘图相关的代码!
  • 只有在这个方法中,我们才能够获取对应的图形上下文[视图相关的图形上下文_绘制的内容可以显示到视图上的]

如果需要图形发生变化,该怎么做?

_重新执行drawRect方法?

  • 程序员禁止直接调用drawRect方法! 原因:直接调用,不能保证上下文被创建好了,会出问题!
  • 如果需要进行重绘,可以通过setNeedsDisplay[常用!] 或者

setNeedsDisplayInRect:方法间接调用!
